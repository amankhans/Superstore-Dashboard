# Superstore Sales Dashboard

## Project Overview

This project analyzes the Superstore Sales dataset using SQL, Laravel, MySQL, and Chart.js. The objective is to identify key business trends, customer behavior, regional performance, and product sales patterns through an interactive dashboard.

## Technologies Used

* PHP Laravel
* MySQL
* Chart.js
* HTML/CSS
* SQL

## Features

* Region Filter
* Category Filter
* Year Filter
* Dynamic KPI Cards
* Year-over-Year Sales Trend
* Monthly Sales Trend
* Regional Sales Analysis
* Category Analysis
* Customer Segment Analysis
* Shipping Mode Analysis
* Average Order Value Analysis
* Top 10 Products Analysis
* Category × Region Matrix

## Business Insights

* West region generated the highest sales.
* Technology category achieved the highest revenue.
* Consumer segment contributed the largest share of sales.
* Sales increased significantly in later years.
* A small number of products generated a large percentage of revenue.
